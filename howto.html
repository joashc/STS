How to use Sequential Trial Simulator

  1. Using the "Add Stage" button and the "x" button on each stage, add or remove stages as required
  2. Enter the number of participants in each stage (or use the up/down arrows)
  3. Enter the number of responses required to pass each stage. Note that this field refers to the TOTAL number of participants who displayed a response, in the entire trial up to this point
  4. The graphs will update in real time as you change these values, allowing you to find an efficient experimental configuration
